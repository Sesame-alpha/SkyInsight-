# SkyInsight-
SkyInsight – African Aviation Intelligence Tool

Real‑time aviation dashboard with ML delay predictions, focused on 8 African airlines.

Features

· Live flights & map (OpenSky Network)
· ML delay predictions (TensorFlow.js)
· Airline performance score (gauge chart)
· Weather at major hubs (Open‑Meteo)
· ETA/ETD table, congestion index
· CSV / PDF export, sound alerts

Tech

HTML, CSS, JS, Leaflet, Chart.js, TensorFlow.js, jsPDF

Airlines tracked

Ethiopian Airlines, South African Airways, EgyptAir, Kenya Airways, Royal Air Maroc, Air Mauritius, TAAG Angola, Air Botswana
